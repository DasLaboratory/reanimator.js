# reanimator.js

Reanimate dead projects faster than you can say Lazarus of Bethany!

## Usage

###### Run reanimator.js

```bash
npx reanimator
# or
npm run reanimator
```

###### Development (watch)

Using [esbuild](https://esbuild.github.io/):

```bash
npm start
# or
npm run watch
# or
npm run dev
```

###### Build

Using [esbuild](https://esbuild.github.io/):

```bash
npm run build
```
